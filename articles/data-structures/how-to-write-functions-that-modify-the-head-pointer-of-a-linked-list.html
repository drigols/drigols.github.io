Data Structures - Linked List


Considere uma representação simples de uma Linked List(Lista vinculada). As funções que operam em tais Linked List podem ser dividias em duas categorias:

  • Funções que não modificam o head da Linked List
  • Funções que modificam o head da Linked List
1) Funções que não modificam o ponteiro head:

Exemplos de funções que não modificam o head de uma Linked List:

  1. Exibir os dados de uma Linked List;
  2. Atualizar os dados de uma Linked List;
  3. Outras operações que acessam dados...

É geralmente fácil de decidir o protótipo de funções desta categoria. Nós sempre podemos passar o head da Linked List como argumento e atravessar/atualizar a Linked List.

Por exemplo, a seguinte função que adiciona um valor inteiro "x" a cada membro de cada Node:

void addXtoList(struct Node *node, int x)
{
   while(node != NULL)
   {
     node->data = node->data + x;
     node = node->next;
    }
}
2) Funções que modificam o ponteiro head

Exemplos de funções que modificam o head de uma Linked List são:

  1. Inserindo um novo Node no início da Linked List (O ponteiro head é sempre modificado nesta função);
  2. Inserindo um novo Node no final (O ponteiro head é modificado somente quando o primeiro Node está sendo inserido);
  3. Excluindo um determinado Node (O ponteiro head é modificado quando o Node excluído é o primeiro Node).

Pode haver diferentes maneiras de atualizar o head de uma Linked List. Porém, vou abordar apenas a maneira que eu acho mais interessante.

Dada uma Linked List, escreva uma função deleteFirst() que exclui o primeiro Node de uma determinada Linked List.

Input:
1 -> 2 -> 3 -> 4
Output:
2 -> 3 -> 4

O algoritmo para resolver o problem é um simples processo de 3 passos:

  1. Armazene o ponteiro do head atual;
  2. Mude o ponteiro head para apontar para o próximo Node (Ou seja, o head vai receber o valor do seu next);
  3. Apague o head anteior.

Use ponteiro para ponteiro (Pointer to Pointer):
Esta abordagem segue a regra C simples: se você quiser modificar a variável local de uma função dentro de outra função, passe o ponteiro para essa variável.

Então, podemos passar o ponteiro para o ponteiro principal (pointer-to-pointer) para modificar o head na nossa função deleteFirst().
